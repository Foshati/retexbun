# React Native, Expo, Tailwind CSS, Bun

## Installation:
* cd retexbun
```line_numbers,js
bun install
```

## Usage:  
```line_numbers,js
bun run start
```

### Using Expo Go
* Press s │ switch to development build
* Press a │ open Android
* Press i │ open iOS simulator
* Press j │ open debugger
* Press r │ reload app
* Press m │ toggle menu
* Press o │ open project code in your editor
